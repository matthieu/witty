Language features:
  * Syntax support for list, hash, regexp
  * Support "let" for locally scoped definitions (of procedures)
  * Tail calls
  * Partial parameters application for currying
  * Pattern matching
  * Homoiconic allowing macros
  * Destructured assignment (?) and in-list assignment (foo[2] = 'bar')
